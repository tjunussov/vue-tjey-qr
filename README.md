## About

**vue-tjey-gauge** TJey Vue.js(2.0) Visual Gauge Plugin

## Usage
import gauge from 'vue-tjey-gauge'
componens: { gauge }
html 
  gauge(:values="values" :spacing="15")

## Params 
- values
- spacing
- colors

## Dependencies
No dependancies
